# Publication Checklist

Before changing the repository from private to public, confirm:

- [ ] Advisor approval has been obtained.
- [ ] The thesis or related manuscript is no longer under a publication restriction.
- [ ] `data/DC-IV.csv` is not tracked by Git.
- [ ] `results/fit/theta_best.csv` is not tracked unless the parameters are approved for release.
- [ ] No employer code, internal parameters, screenshots, or documents are present.
- [ ] No collaborator-owned data are included without written permission.
- [ ] All sample figures use public, synthetic, or approved data.
- [ ] The README accurately distinguishes measured results from demonstrations.
- [ ] Repository history has been checked for accidentally committed sensitive files.
